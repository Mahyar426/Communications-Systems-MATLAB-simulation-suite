This repository contains exercises, notes, and assignments from the Communication Systems course at Politecnico di Torino, focusing on key aspects of digital communications, channel coding, modulation, error correction, and system analysis.

Repository Contents

Assignment 1

Error detection using Generator and Parity Check matrices

Generator polynomial and code properties

Cyclic Redundancy Check (CRC) design and performance evaluation

Assignment 2

Hard correction vs. Soft correction decoding methods

Linear Feedback Shift Registers (LFSR) and m-sequences

Synchronization marker detection and analysis

Autocorrelation properties and FFT analysis

Assignment 3

Gold sequences generation and their correlation properties

Code Division Multiple Access (CDMA) system analysis

Hartogs-Hughes algorithm for OFDM bit allocation

Satellite Communication

Satellite link budget analysis for LEO satellites

Channel modeling: Free Space Path Loss, Doppler effect, and multipath propagation

Ground segment considerations, including antennas and RF front-ends

Regulations, interference management, and system optimization

Learning Objectives

Master concepts in digital communication systems

Implement and simulate modulation, coding, and communication protocols

Analyze communication system performance, including error rates and signal integrity

Gain practical insights into satellite communication and associated challenges

Tools and Software

MATLAB for simulations and graphical analysis

Satellite Communications Toolbox for advanced system simulations
